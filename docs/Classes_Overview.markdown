 This page lists all the different species and their abilities that differentiate them from another. Some of these are relevant for the [general character creation](/) and [species](/species/).

There are 12 different classes in the Dungeons and Dragons 5e 2024 Player Handbook. 

- Barbarian
- Bard
- Cleric
- Druid
- Fighter
- Monk
- Paladin
- Ranger
- Rogue 
- Sorcerer
- Warlock
- Wizard
